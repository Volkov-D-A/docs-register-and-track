1.0.0   2026-03-29
    - Изменение полей в карточке входящего документа
        Убраны поля оргнизация получатель и адресат, добавлены поля автор резолюции и исполнители резолюции, поля краткое содержание и содержание объединены в одно поле содержание. 
    - Окно "О программе"
        Содержит информацию о текущей версии программы, список изменений в текущей версии, также реализован функционал показа изменений при первом запуске после обновления.
    - Блокировка пользователей при множественных попытках входа
        Добавлен механизм защиты от подбора пароля, при вводе неверного пароля более 3 раз подряд, пользователь блокируется. Блокировка может быть снята только администратором.
